## Student Record Management System

This project implements a Student Record Management System using
Binary Search Tree (BST) in Java.

### Features
- Insert, search, delete student records
- BST traversals (inorder, preorder, postorder)
- Sorting using Merge Sort (by name)
- Sorting using Quick Sort (by CGPA)
- CSV file loading
- Performance testing

### How to Run
1. Open the project in a Java IDE
2. Run `Main.java`
3. Ensure CSV file is in the correct directory
"# student_Record_Management_System"
